## TCGA.qlui

TCGA.qlui is a GUI for [TCGA.ql](https://github.com/drobbins/TCGA.ql), built as a component for the [TCGA Toolbox](https://github.com/tcga/web-toolbox).

[Load TCGA.qlui in TCGA Toolbox](http://tcga.github.com/?module=https://raw.github.com/drobbins/TCGA.qlui/master/tcgaqlui.js&module=https://dl.dropbox.com/u/4000409/Repos/TCGA.ql/tcgaql.js&module=https://ajax.googleapis.com/ajax/libs/angularjs/1.0.3/angular.min.js).
